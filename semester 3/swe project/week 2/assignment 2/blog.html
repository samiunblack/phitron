Discuss the scope of var, let and const

var

  • Scope: Variables declared with var keyword are only accessible in the function they are declared
  • Hoisting: Variables declared with var keyword can be used before the real declaration, but it's value will be undefined till the real declaration

let

  • Scope: Variables declared with let keyword are accessible in the block they are declared, it can be anything defined with curly braces
  • Hoisting: Variables declared with let keyword cannot be used before their real declaration

const

  • Scope: Variables declared with const keywords are also block scoped
  • Immutability: Once a const variable is declared it can never be reassigned

Tell us the use cases of null and undefined

null

  • null keyword shows the initial absence of value. It is something that is used when we want to keep something as empty
  • null keyword can be returned by a function to indicated they didn't find the desired result

undefined

  • when a variable is declared but not assigned, javascript puts undefined as its value automatically
  • if a function doesnt return any value it returns undefined
  • when you try to access a property of any element that does not exist you get undefined

What do you mean by REST API?

A REST API is an API that follows the design principles of represented state transfer architectural style. Web APIs that follows the REST framework are known as REST API.
